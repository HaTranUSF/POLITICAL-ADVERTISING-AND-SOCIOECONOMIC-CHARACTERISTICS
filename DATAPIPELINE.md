# Notebooks
 
Run these in order to reproduce the full data pipeline.
 
| Notebook | Description |
|---|---|
| `01_data_cleaning.ipynb` | Cleans raw Census ACS files for income, poverty, education, and population |
| `02_data_merge.ipynb` | Creates the State_Year_Key bridge and audits merge coverage |
| `03_master_pipeline.ipynb` | Assembles the final socioeconomic panel with state abbreviation mapping |
